# Speedup Services

This project contains the Speedup Services marketing site built with React, Vite, and Tailwind CSS.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
3. Build for production:
   `npm run build`
